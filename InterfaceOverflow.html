Interface Overflow

Video Game made by Unreal Engine 5 | 2025

Interface Overflow is a work-in-progress game that turns UI, an element often treated as invisible background support, into the main protagonist. Instead of using UI simply to explain a world, the world itself is built from interfaces. The screen becomes the space you navigate, and the interface actively shapes what you notice, believe, and do.

So far, the project includes only the first level, designed as a starting point for a larger series. It introduces the core idea by pushing the player into a space of saturation and interference, where navigation and decision-making are constantly redirected by the interface. Upcoming levels will extend these experiments, such as printing the player's keyboard inputs directly onto the screen. Updates and new builds will be posted on my website.
